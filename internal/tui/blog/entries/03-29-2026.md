Integration Options
- Legacy: Collecting card token from client
- Modernized: Generating a client secret for client

Options for Mordernized Checkout:
- Stripe Hosted
- Embedded (Checkout Session API): Pre-styled (non-customizable)
- Custom Embedded (Checkout Session API): Limited customization
- Payment Element (PaymentIntent API): Lowest-level API with full customization support, even with enabled payment methos (i.e., "Pay with Link")
