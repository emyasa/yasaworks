# Go: Concurrency

> Monday 12:08 AM

Two key concepts: goroutines and channels. With goroutines, any function (including function literals) can be run concurrently by prefixing the call with the `go` keyword. Minimal example:
```
func main() {
	go func() {
		fmt.Println("here")
	}()

	time.Sleep(1 * time.Second) 
}
```
Note: The sleep is just to get it printed to standard output before the program exits. A WaitGroup should be useful for waiting on multiple goroutines.
Channels, as the name suggests, serve as a means of communication between goroutines, which provides a way to listen for a goroutine's completion. A blocking unbuffered channel example:
```
func main() {
	ch := make(chan struct{})
	go func() {
		<- ch // blocking until it receives a value
		fmt.Println("done")
	}()

	ch <- struct{}{} 
    // ^sends a value into the channel. 
    // ^It waits until the value is set
}
```
The select statement is similar to a switch statement, but specifically for channels. 
It becomes blocking when there's no default case, so given one or more cases, it waits until at least one channel operation (either send or a receive) can proceed:
```
func main() {
	ch1 := make(chan int)
	go func() {
		ch1 <- 1
	}()
	ch2 := make(chan int)
	go func() {
		ch2 <- 1
	}()

	select {
	case <- ch1:
		fmt.Println("ch1")
	case <- ch2:
		fmt.Println("ch2")
	}
}
```

For a non-blocking select statement, a default case can be added. This allows unready unbuffered channels to be ignored, preventing unnecessary waiting. Another option is to use buffered channel, which can receive values right away as long as it is not full. A fan-out example should help demonstrate this clearly:
```
type client struct {
	name string
	send chan struct{}
}

func (c client) setup() {
	go func() {
		for {
			<- c.send
			fmt.Println("received in " + c.name)
		}
	}()
}

func broadcast(registry []client) {
	for _, client := range registry {
		select {
		case client.send <- struct{}{}:
		default:
		}
	}
}

func main() {
	c1 := client{
        name: "c1", 
        send: make(chan struct{}, 1),
    }
	c1.setup()

	c2 := client{
        name: "c2", 
        send: make(chan struct{}, 1),
    }
	c2.setup()

	registry := []client{c1, c2}
	broadcast(registry)

	time.Sleep(1 * time.Second)
	broadcast(registry)

	time.Sleep(1 * time.Second)
}
```
